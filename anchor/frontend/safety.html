About this view

Real-time location tracking requires the Anchor mobile companion (iOS/Android), which is out of scope for this hackathon prototype. This page shows what a carer would see with the mobile app paired:

  • Live GPS position updated every 30 seconds
  • Geofenced safe-zone breach alerts (wandering detection)
  • Last-seen timestamp if connectivity drops

The simulate button below fires the same carer escalation that a real safe-zone breach would — useful for demoing the alert chain end-to-end.
